# Tirepick ROOT web (Demo)

## 0. ENV

```
  next.js + react.js
  styled-components
  react-query
  typescript
  nprogress
  moment
  jotai
```

## 1. directory

```
  /public
  /src
    - /api        : fetch function
    - /assets     : build data (icons..)
    - /components : react compoenent
    - /contants   : CONST
    - /hooks      : react custom hooks
    - /models     : viewmodel
    - /pages      : next page components
    - /store      : jotai store
    - /utils      : util components
  **
```
